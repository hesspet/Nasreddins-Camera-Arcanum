# Foto gespeichert

## Speicher-Bestätigung

Nachdem du ein Foto aufgenommen oder hochgeladen hast, zeigt die App an, wo das Bild gespeichert wurde.

## Zwei Fälle

- **"Foto gespeichert unter …"**: Das Bild wurde dauerhaft auf deinem Gerät gespeichert
  (unterstützt von modernen Browsern mit File System Access API).
- **"Foto im Browser zwischengespeichert …"**: Das Bild wird im flüchtigen Browserspeicher gehalten.
  Es bleibt verfügbar, solange die App geöffnet ist.

## Was bedeutet das für dich?

In beiden Fällen kannst du das Bild weiterverarbeiten (Person freistellen, Geisterebene einfügen, herunterladen).
Der Speicherort beeinflusst nur, ob das Originalbild nach einem Neuladen der App noch verfügbar ist.
